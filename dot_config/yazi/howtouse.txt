  Yazi 文件管理器使用指南
  1. 基础操作
  启动和基本界面
  - Yazi 是一个现代化的终端文件管理器，采用类似 vim 的操作方式
  - 启动命令：yazi 或 ya
  - 界面分为三栏：左侧导航栏、中间文件列表、右侧预览栏

  基本导航
  - j 或 ← - 返回上级目录
  - l 或 → - 进入选中的目录
  - <S-Left> - 回退到上一个目录
  - <S-Right> - 前进到下一个目录
  - g + g - 跳转到列表顶部
  - g + t - 跳转到列表底部

  光标移动
  - i 或 ↑ - 上一个文件
  - k 或 ↓ - 下一个文件
  - <C-i> 或 <PageUp> - 向上移动半页
  - <C-k> 或 <PageDown> - 向下移动半页
  - <S-i> 或 <S-PageUp> - 向上移动一页
  - <S-k> 或 <S-PageDown> - 向下移动一页

  2. 快速目录跳转
  根据你的配置，有以下快速跳转功能：

  - g + h - 跳转到主目录 (~)
  - g + d - 跳转到下载目录 (~/Downloads/)
  - g + p - 跳转到图片目录 (~/Pictures/)
  - g + v - 跳转到视频目录 (~/Videos/)
  - g + m - 跳转到音乐目录 (~/Music/)
  - g + w - 跳转到文档目录 (~/Documents/)
  - g + u - 跳转到公共目录 (~/Public/)
  - g + t - 跳转到模板目录 (~/Templates/)
  - g + <Space> - 交互式目录跳转

  3. 文件操作
  文件选择
  - <Space> - 切换当前文件的选择状态
  - <C-a> - 选择所有文件
  - <C-r> - 反选所有文件
  - v - 进入视觉模式（选择多个连续文件）
  - V - 进入视觉模式（取消选择）

  复制粘贴
  - y - 复制选定文件
  - x - 剪切选定文件
  - p + p - 粘贴复制/剪切的文件
  - p + f - 强制粘贴（覆盖现有文件）
  - Y 或 X - 取消复制/剪切状态

  文件创建与删除
  - a - 创建文件/目录（以 / 结尾创建目录）
  - d - 移动到回收站（使用回收站插件）
  - D - 永久删除文件
  - r + r - 重命名文件或目录（光标在扩展名前）
  - r + f - 重命名文件并清空整个文件名
  - r + i - 重命名文件（光标在文件名开头）
  - r + a - 重命名文件（光标在文件名末尾）

  打开文件
  - o - 打开选定文件
  - O - 交互式打开选定文件
  - <Enter> - 打开选定文件
  - <S-Enter> - 交互式打开选定文件

  4. 高级功能
  符号链接
  - - - 创建符号链接（绝对路径）
  - _ - 创建符号链接（相对路径）
  - <C--> - 创建硬链接
  - g + f - 跟随符号链接

  复制路径
  - c + c - 复制文件路径
  - c + d - 复制目录路径
  - c + f - 复制文件名
  - c + n - 复制文件名（不含扩展名）

  5. 搜索和过滤
  搜索功能
  - f - 智能过滤文件
  - / - 智能查找下一个文件
  - ? - 智能查找上一个文件
  - n - 下一个匹配项
  - N - 上一个匹配项
  - s - 通过 fd 搜索文件名
  - S - 通过 ripgrep 搜索文件内容
  - z - 通过 fzf 跳转文件/目录
  - Z - 通过 zoxide 跳转目录

  6. 排序和显示
  排序设置
  - , + m - 按修改时间排序（正序）
  - , + M - 按修改时间排序（倒序）
  - , + b - 按创建时间排序（正序）
  - , + B - 按创建时间排序（倒序）
  - , + e - 按扩展名排序（正序）
  - , + E - 按扩展名排序（倒序）
  - , + a - 按字母顺序排序（正序）
  - , + A - 按字母顺序排序（倒序）
  - , + n - 自然排序（正序）
  - , + N - 自然排序（倒序）
  - , + s - 按大小排序（正序）
  - , + S - 按大小排序（倒序）
  - , + r - 随机排序

  显示设置
  - . - 切换隐藏文件显示
  - m + s - 行模式：显示大小
  - m + p - 行模式：显示权限
  - m + b - 行模式：显示创建时间
  - m + m - 行模式：显示修改时间
  - m + o - 行模式：显示所有者
  - m + n - 行模式：无信息

  7. 标签页管理
  - t - 创建新标签页（使用当前目录）
  - 1 - 切换到第一个标签页
  - 2 - 切换到第二个标签页
  - ...
  - 9 - 切换到第九个标签页
  - - - 切换到上一个标签页
  - = - 切换到下一个标签页
  - _ - 与上一个标签页交换位置
  - + - 与下一个标签页交换位置

  8. 插件功能
  书签插件 (whoosh)
  - [ - 通过书签快捷键跳转
  - } - 直接模糊搜索书签
  - ] + a - 添加书签（当前文件/目录）
  - ] + A - 添加书签（当前目录）
  - ] + t - 添加临时书签（当前文件/目录）
  - ] + T - 添加临时书签（当前目录）
  - <A-k> - 直接跳转到书签 'k'
  - ] + f - 通过模糊搜索跳转到书签
  - ] + d - 通过快捷键删除书签
  - ] + D - 通过模糊搜索删除书签
  - ] + C - 删除所有用户书签
  - ] + r - 通过快捷键重命名书签
  - ] + R - 通过模糊搜索重命名书签

  回收站插件
  - R + b - 回收站管理

  压缩插件 (ouch)
  - c + z - 使用 ouch 进行压缩

  GVFS 插件
  - M + m - 选择设备并挂载并跳转
  - M + R - 重新挂载当前目录的设备
  - M + u - 卸载并弹出设备
  - M + U - 强制卸载并弹出设备
  - M + a - 添加挂载 URI
  - M + e - 编辑挂载 URI
  - M + r - 移除挂载 URI
  - G + m - 跳转到设备挂载点
  - ` + ` - 跳回上一个目录

  挂载插件
  - M + p - 挂载硬盘

  9. 任务和系统操作
  任务管理
  - w - 显示任务管理器

  系统操作
  - <Esc> - 退出视觉模式、清除选择或取消搜索
  - q - 退出程序
  - Q - 退出且不输出 cwd 文件
  - <C-c> - 关闭当前标签页，如是最后一个则退出
  - <C-z> - 挂起程序
  - ~ 或 <C-F1> - 打开帮助

  10. 配置说明
  根据你的配置文件，你启用了以下特性：

  - 三栏布局比例为 1:4:3
  - 按扩展名排序
  - 不区分大小写排序
  - 不反转排序
  - 目录优先显示
  - 行模式显示创建时间
  - 显示隐藏文件
  - 显示符号链接
  - 预览窗口自动换行
  - 使用 Dracula 主题

  11. 特殊插件功能
  你的配置还包括以下特殊插件功能：

  - mediainfo 插件：为音频、视频、图像文件提供详细信息预览
  - duckdb 插件：支持预览 CSV、TSV、JSON、Parquet、XLSX 等数据文件
  - bypass 插件：智能进入功能
  - full-border 插件：完整的边框显示
  - starship 插件：集成 Starship 提示符
  - simple-status 插件：简化状态栏

  以上就是基于你的配置文件的 yazi 详细使用指南。这些快捷键和功能将大大提高你在终端中的文件管理效率。

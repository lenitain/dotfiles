# Tmux 配置使用文档

## 概述

这是一个高度定制化的 tmux 配置，具有以下特点：
- 会话管理和自动编号
- 面板导航和布局管理
- 与系统剪贴板集成
- 丰富的状态栏显示
- 与 agent-tracker 集成（任务跟踪）
- FZF 面板选择器

## 前缀键

- **前缀键**: `Ctrl-s` (默认 `Ctrl-b` 已解绑)

## 会话管理

### 创建会话
- `Ctrl-s + C-c`: 创建新会话
- `Alt + Shift + S`: 通过脚本创建新会话

### 切换会话
- `Ctrl + 1-9`: 切换到对应编号的会话
- `F1-F5`: 切换到对应编号的会话
- `Ctrl-s + W`: 显示会话/窗口选择树

### 移动会话
- `Ctrl-s + l`: 将当前会话向左移动
- `Ctrl-s + y`: 将当前会话向右移动

### 重命名会话
- `Ctrl-s + .`: 重命名会话（会自动重新编号）

## 窗口管理

### 创建/关闭窗口
- `Alt + O`: 在当前路径创建新窗口
- `Alt + Q`: 关闭当前面板

### 切换窗口
- `Ctrl-s + Ctrl-p`: 上一个窗口
- `Ctrl-s + Ctrl-n`: 下一个窗口
- `Alt + 1-9`: 切换到指定编号的窗口

### 重命名窗口
- `Ctrl-s + ,`: 重命名当前窗口

### 移动窗口到会话
- `Ctrl-s + 1-9`: 将当前窗口移动到对应编号的会话

### 交换窗口位置
- `Alt + L`: 将窗口向左交换
- `Alt + Y`: 将窗口向右交换

## 面板管理

### 分割面板
- `u`: 向上垂直分割面板
- `e`: 向下垂直分割面板
- `n`: 向左水平分割面板
- `i`: 向右水平分割面板

### 面板导航
- `Alt + n`: 移动到左侧面板
- `Alt + e`: 移动到下方面板
- `Alt + u`: 移动到上方面板
- `Alt + i`: 移动到右侧面板

### 面板交换
- `>`: 向下交换面板
- `<`: 向上交换面板
- `|`: 交换面板

### 面板调整大小
- `Alt + Shift + N`: 向左调整面板宽度（3格）
- `Alt + Shift + E`: 向下调整面板高度（3格）
- `Alt + Shift + U`: 向上调整面板高度（3格）
- `Alt + Shift + I`: 向右调整面板宽度（3格）

### 面板操作
- `Alt + F`: 全屏/取消全屏当前面板
- `Alt + O`: 将当前面板移出新窗口
- `Space`: 切换面板布局（水平/垂直）
- `Alt + b`: 切换未读标记
- `Alt + w`: 切换监视标记

### 将面板移动到指定窗口
- `Alt + Shift + 1-9`: 将当前面板移动到对应编号的窗口

## 复制模式

### 进入复制模式
- `Alt + v`: 进入复制模式

### 复制模式中的导航（Vi 模式）
- `n/i/u/e`: 左/右/上/下移动光标
- `N/I`: 移动到行首/行尾
- `h`: 移动到下一个单词结尾
- `v`: 开始选择
- `Ctrl + v`: 切换矩形选择
- `y`: 复制到系统剪贴板
- `Y`: 复制到行尾
- `Ctrl + u`: 向上滚动5行
- `Ctrl + e`: 向下滚动5行

### 粘贴
- `Ctrl + Shift + v`: 从系统剪贴板粘贴
- `Alt + Shift + V`: 从系统剪贴板粘贴

## 布局管理

- `Ctrl-s + I`: 快速创建右侧布局
- `Ctrl-s + N`: 快速创建左侧布局
- `Ctrl-s + U`: 快速创建上方布局
- `Ctrl-s + E`: 快速创建下方布局

## 其他功能

### 同步输入
- `Ctrl-s + Ctrl-g`: 切换所有面板同步输入（开启时边框变红）

### FZF 面板选择器
- 通过 `fzf_panes.tmux` 提供 MRU（最近使用）面板选择
- 支持多选面板并自动排列
- 按 `Alt+p` 切换预览
- 按 `Ctrl+r` 刷新
- 按 `Ctrl+x` 删除面板
- 按 `Ctrl+v` 水平移动面板
- 按 `Ctrl+s` 垂直移动面板
- 按 `Ctrl+t` 交换面板

### 缓冲区管理
- `Ctrl-s + b`: 列出所有缓冲区
- `Ctrl-s + p`: 粘贴缓冲区

### 重新加载配置
- `Ctrl-s + r`: 重新加载配置文件

## 状态栏

### 左侧状态栏
- 显示所有会话（带编号）
- 当前会话高亮显示
- 窄屏模式下只显示编号
- 任务图标集成：
  - `⏳`: 进行中的任务
  - `🔔`: 未完成任务或未读标记

### 右侧状态栏
- Rainbarf CPU/系统负载图表（可配置）
- 当前面板内存使用
- 当前窗口内存使用
- 总内存使用
- Notes 计数
- 主机名

### 窗口状态
- 显示窗口名称
- 任务图标集成
- 当前窗口高亮显示

## 面板边框

- 激活面板显示详细状态（包括 starship 提示符）
- 非激活面板显示简化状态
- 全屏时显示 `⛶` 图标

## 集成功能

### Agent-Tracker 集成
- 自动通知任务状态
- `Alt + t`: 切换任务跟踪
- `Alt + m`: 聚焦最新通知
- `Alt + Shift + M`: 聚焦最后源

### 项目特定钩子
- 当切换窗口时，自动检查并运行 `./on-tmux-window-activate.sh` 或 `../on-tmux-window-activate.sh`

### TPM 插件管理
- 已安装插件：
  - `tmux-plugins/tpm`: 插件管理器
  - `tmux-plugins/tmux-resurrect`: 会话恢复
  - `tmux-plugins/tmux-continuum`: 自动保存
- 自动每5分钟保存会话
- 按 `Alt + F12` 管理插件（安装/更新/清理）

## 主题

- 支持真彩色
- 可通过 `TMUX_THEME_COLOR` 环境变量自定义主题色
- 默认主题色: `#b294bb` (紫色)

## 环境变量

配置会自动同步以下环境变量到新会话：
- `DISPLAY`
- `DBUS_SESSION_BUS_ADDRESS`
- `QT_IM_MODULE`, `QT_QPA_PLATFORMTHEME`
- `XDG_*` 系列变量
- `FZF_DEFAULT_OPTS`
- `TMUX_THEME_COLOR`
- `TERM`, `TERM_PROGRAM`

## 注意事项

1. 确保已安装依赖：`fzf`, `python3`, `starship`
2. 首次使用需安装 TPM 插件：按 `Alt+F12` 或手动运行 `~/.tmux/plugins/tpm/tpm`
3. 如需使用 agent-tracker 功能，需先安装 agent-tracker
4. 状态栏宽度小于 80 字符时会自动隐藏部分信息
